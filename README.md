# 🌱 Farm2Home - E-Commerce Platform

## 📖 Overview

Farm2Home is a web-based e-commerce platform that connects farmers directly with customers. The platform eliminates unnecessary intermediaries, enabling farmers to sell fresh agricultural products at fair prices while customers receive high-quality produce at affordable rates.

The system provides a simple, secure, and user-friendly shopping experience with features such as product browsing, shopping cart, online ordering, and order management.

---

# 🎯 Objectives

* Connect farmers directly with customers.
* Provide fresh and quality agricultural products.
* Ensure fair pricing for both farmers and customers.
* Simplify online shopping for farm products.
* Reduce dependency on middlemen.

# 🚀 Installation

1. Clone the repository

```
git clone https://github.com/your-username/Farm2Home.git
```

2. Move into the project directory

```
cd Farm2Home
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Configure the MySQL database.

5. Import the `farm2home.sql` file.

6. Start the Flask server

```
python app.py
```

7. Open your browser

```
http://localhost:5000
```
